Trang web quản lý trung tâm mỹ thuật sáng tạo

Hệ thống quản lý trung tâm mỹ thuật toàn diện với Django framework

12/2024 - 02/2025
2 tháng
Individual Project

Tổng quan dự án

Xây dựng website quản lý hoạt động của trung tâm mỹ thuật với đầy đủ tính năng quản lý học viên và giảng viên, thêm, sửa, xóa và tạo báo cáo có dữ liệu. Thiết kế back-end, phát triển RESTful API quản lý khóa học, học viên, giảng viên và triển khai cơ sở dữ liệu PostgreSQL trên Aiven Cloud. Hoàn thành và triển khai dự án trong 2 tháng với tối ưu hiệu năng và đảm bảo khả năng mở rộng hệ thống.

Mục tiêu dự án

  • Quản lý học viên và giảng viên: thêm, sửa, xóa và thống kê
  • Theo dõi đăng ký và tạo báo cáo có dữ liệu
  • Tìm kiếm khóa học, quản lý tài liệu học tập
  • Giao diện quản trị thân thiện với Django Admin
  • Thiết kế back-end và phát triển RESTful API
  • Triển khai cơ sở dữ liệu PostgreSQL trên Aiven Cloud
  • Kiểm thử, debug và tích hợp API với giao diện người dùng

Vai trò của tôi

  • Full-stack Developer: Phát triển cả frontend và backend với Django
  • Database Architect: Thiết kế PostgreSQL schema trên Aiven Cloud
  • API Developer: Xây dựng RESTful API cho quản lý hệ thống
  • QA Engineer: Kiểm thử, debug và tối ưu hiệu năng

Tech Stack

Python Django PostgreSQL Aiven Cloud REST API PgAdmin 4 Git HTML/CSS JavaScript Bootstrap

Quy trình thực hiện

01

System Analysis & Database Design

Phân tích yêu cầu hệ thống quản lý trung tâm mỹ thuật, thiết kế database schema và define các relationships. Lập kế hoạch triển khai PostgreSQL trên Aiven Cloud để đảm bảo hiệu năng và khả năng mở rộng.

• Requirements analysis • PostgreSQL schema design • Aiven Cloud setup
02

Backend Development & API

Phát triển Django backend với models cho học viên, giảng viên, khóa học. Xây dựng RESTful API để quản lý CRUD operations, authentication và authorization. Tích hợp Django Admin interface.

• Django Models & Admin • RESTful API development • User authentication system
03

Frontend Development & Reports

Tạo giao diện người dùng với HTML/CSS/JavaScript và Bootstrap. Phát triển tính năng tìm kiếm khóa học, quản lý tài liệu học tập và tạo báo cáo có dữ liệu thống kê.

• Responsive UI with Bootstrap • Course search functionality • Data reports & statistics
04

Testing & Deployment

Thực hiện kiểm thử toàn diện, debug và tối ưu hiệu năng hệ thống. Tích hợp API với giao diện người dùng, đảm bảo data consistency và triển khai thành công trong 2 tháng.

• Comprehensive testing • Performance optimization • API integration & deployment
