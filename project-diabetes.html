Phân loại bệnh tiểu đường qua trên phương pháp học máy

Xây dựng hệ thống dự đoán nguy cơ mắc bệnh tiểu đường sử dụng Machine Learning

03/2025 - 06/2025
3 tháng
Individual Project

Tổng quan dự án

Dự án nghiên cứu và phát triển hệ thống phân loại bệnh tiểu đường sử dụng dữ liệu y tế từ bộ dữ liệu Pima Indians Diabetes. Hệ thống áp dụng các kỹ thuật tiền xử lý dữ liệu, tạo đặc trưng mới, xử lý mất cân bằng dữ liệu bằng SMOTE và RandomUnderSampler, và sử dụng 4 mô hình học máy để đạt độ chính xác cao nhất.

Mục tiêu dự án

  • Xây dựng mô hình ML với F1-score đạt 0.78
  • Áp dụng 4 thuật toán: Logistic Regression, Decision Tree, Random Forest, KNN
  • Tối ưu thành 5 mô hình bằng GridSearchCV
  • Đánh giá mô hình bằng Accuracy, Precision, Recall, F1-score, ROC-AUC
  • Triển khai mô hình tốt nhất lên ứng dụng web Flask/Python

Vai trò của tôi

  • Data Scientist: Thu thập, làm sạch và phân tích dữ liệu Pima Indians
  • ML Engineer: Xây dựng và tối ưu hóa 4 mô hình học máy
  • Backend Developer: Phát triển API Flask/Python
  • Research: Nghiên cứu và đánh giá hiệu năng các mô hình

Tech Stack

Python Scikit-learn Pandas NumPy SMOTE RandomUnderSampler Logistic Regression Decision Tree Random Forest KNN GridSearchCV Flask

Quy trình thực hiện

01

Thu thập và khám phá dữ liệu

Sử dụng dataset Pima Indians Diabetes với 768 mẫu dữ liệu và 8 đặc trưng. Tiền xử lý dữ liệu, tạo đặc trưng mới, xử lý mất cân bằng bằng SMOTE và RandomUnderSampler để cải thiện hiệu năng mô hình.

• 768 samples, 8 features • SMOTE & RandomUnderSampler • Feature engineering
02

Tiền xử lý dữ liệu

Làm sạch dữ liệu, xử lý missing values, chuẩn hóa features và feature scaling. Thực hiện feature engineering để tạo ra các features mới có ý nghĩa.

• StandardScaler normalization • Feature engineering • Train/test split (80/20)
03

Xây dựng và huấn luyện mô hình

Áp dụng 4 thuật toán machine learning: Logistic Regression, Decision Tree, Random Forest, và KNN. Tối ưu hóa thành 5 mô hình bằng GridSearchCV để tìm hyperparameters tốt nhất.

• 4 ML algorithms: LR, DT, RF, KNN • GridSearchCV optimization • 5 optimized models
04

Đánh giá và triển khai mô hình

Đánh giá mô hình bằng các chỉ số: Accuracy, Precision, Recall, F1-score, ROC-AUC. Triển khai mô hình tốt nhất lên ứng dụng web sử dụng Flask/Python để dự đoán dữ liệu đầu vào và hiển thị kết quả.

• F1-score: 0.78 • Flask web application • Real-time prediction
